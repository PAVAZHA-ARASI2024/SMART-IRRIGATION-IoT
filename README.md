
# Smart Irrigation System using IoT

## Overview
This project is an IoT-based smart irrigation system that automates watering based on real-time soil moisture levels. It helps conserve water and ensures optimal plant health. The system uses sensors to monitor soil conditions and a Weather API to track climate changes, triggering irrigation only when needed. Designed for efficient farming, it reduces manual effort and prevents overwatering.

## Features
- Monitors soil moisture in real time  
- Automates irrigation to save water  
- Improves crop health with efficient watering  
- Reduces manual effort
